# Team 7 Final Project GitHub repository
This repository contains the entire codebase, related to this project. 
The highest-scoring model is contained within the modelisation.ipynb Jupyter notebook; running it requires downloading the notebook into a suitable environment and ensuring that the dataset is put within the initially listed directory. After that, the notebook should be able to run, given that all th required dependencies ahve been installed.

exploration.ipynb is a notebook, containing the initial exploaration of the dataset, engineering initial features, and a correlation study on which features are the most linked to score.

the TFT+LightGBM attempts folder contains TFT_notebook.ipynb and extra dependency files/auxiliary scripts for it to run. It was intended as an architecrure upgrade fo the XGBoost solution, but fell short, with the LightGBM implementation scoring as the second highest scoring arhictecture of our team made to solve the provided task, with the TFT architecture having the lowest architecture score result. TFT_notebook.ipynb requires to be downloaded to be viewed, and is recommended to run only inside its fully downloaded parent folder.
